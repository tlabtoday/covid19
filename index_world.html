"The current state of COVID-19 data"

This web site focus on the visualization of COVID-19 data. We started to visualize "the current state" using basic techniques in Tanaka Laboratory.

These charts were plotted using "COVID-19 Data Repository by Johns Hopkins CSSE", "Data of COVID-19 Infections in Japan(Ministry of Health, Labor and Welfare)", and Tableau Public. Thank you for public data by Johns Hopkins CSSE, Ministry of Health, Labor and Welfare(Japan), and BI tool, Tableau, by Tableau Software, LLC.
