"The current state of COVID-19 data"

This web site focus on the visualization of COVID-19 data. We started to visualize "the current state" using basic techniques in Tanaka Laboratory.

These charts were plotted using "COVID-19 Data Repository by Johns Hopkins CSSE" and Tableau Public. Thank you for public data by Johns Hopkins CSSE and BI tool, Tableau, by Tableau Software, LLC.
